Get a variable from a .env file

I went browsing around to see if this is somehow possible with emacs lisp, but seems that this is more difficult that I've would've anticipated.

So let's see if with a bit of shell we can get our luck. 

grep "${key}=" < ../../../.env| cut -d '=' -f2
